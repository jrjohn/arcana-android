Privacy Policy

Last updated: March 17, 2026

This Privacy Policy describes how Arcana (com.arcana.example) handles your information. The app is developed by Jr. John (jr.johnchang@gmail.com).

1. Information We Collect

Arcana is an architecture reference application. It does not collect, store, or transmit any personally identifiable information. The app communicates with a mock API endpoint (reqres.in) for demonstration purposes only.

2. Data Storage

Any data cached by the app is kept locally on your device using Room database. This data is never shared with third parties and is removed when you uninstall the app.

3. Third-Party Services

  • reqres.in — mock REST API (no personal data submitted)

4. Permissions

Arcana requests only the INTERNET permission, required to communicate with the demo API. No camera, microphone, contacts, or location permissions are requested.

5. Children's Privacy

This application is not directed at children under the age of 13.
